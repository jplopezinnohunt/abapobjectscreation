Common Pitfalls

1
T028B forgotten — T035D and T028B are in the same SPRO screen but different tables. Both must be maintained or EBS posting fails silently.
2
KTOKS = OTHR — When copying from a reference bank, the account group may be OTHR instead of BANK. Must be corrected in SKA1.
3
OBA1 incomplete for non-USD — All three fields (LKORR, LSBEW, LHBEW) must be filled for foreign-currency clearing accounts. Skipping the valuation section causes month-end failures.
4
XINTB flag missing — The "interest indicator" flag in SKB1 must be set for both 10* and 11* accounts. Missed when creating manually.
5
GS02 sets not updated — YBANK sets are maintained manually per system. New accounts must be added to the correct set node or average balance reports will be incomplete.
6
IBAN in Workbench transport — TIBAN entries travel via Workbench transport (not Customizing), which requires a different transport request type.
7
G/L created only in PRD — Master Data team creates G/L accounts in P01 only. D01 copies must be created via RFC INSERT or manual FS00.

The Two Tables: T035D and T028B

Key Insight: T035D and T028B are configured in the SAME SPRO screen (SM30 V_T035D) but are different tables with different keys. Both are required for EBS to work. Forgetting T028B is the #1 configuration error.

T035D — Account Determination

Field Description Example
DISKB Cash Mgmt name UBA01-USD1
BNKKO G/L account 1065421

Maps the Cash Management name to the bank G/L account for posting.

T028B — Bank Posting Rules

Field Description Example
BANKL Bank key 000400122
KTONR Bank account # 36061234
VGTYP Format variant XRT940
BNKKO Cash Mgmt name UBA01-USD1

Maps the physical bank key + account number to the format variant and Cash Management name.

How They Connect

T028B
BANKL + KTONR → BNKKO (CM name)
BNKKO = DISKB →
T035D
DISKB (CM name) → BNKKO (G/L)

The BNKKO field in T028B is the Cash Management name (e.g., UBA01-USD1), which matches DISKB in T035D. The BNKKO field in T035D is the G/L account number (e.g., 1065421). Same field name, different meaning in each table.
